Web Job Keywords

On August 27, 2015, I gathered the results from three searches on GlassDoor in an attempt to define the skillset wanted by Portland employers for the following terms:

  • Web Developer
  • Backend Web Developer
  • Backend Developer

Not Scientific

The "Web Developer" search had 7 pages of entries, but I only included the first one. When looking at totals, please remember that these are simply the number of times the words were listed, not a qualitative analysis for true comparisons. The results were pretty obvious without including the other 6 pages of Frontend Web Developer listings.The two Backend Web Developer results were duplicates of Frontend Web Developer listings and were not counted. The Backend Developer results had a total of 6 unique listings, not pages, which focused mostly on JAVA. After removing all of the duplicates, here are the results organized in various ways.

I've listed each keyword, the number of times it occurred, and how it relates to other keywords.
